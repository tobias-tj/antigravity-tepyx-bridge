# Antigravity Tepyx Bridge

**Antigravity Tepyx Bridge** es la extensión oficial de VS Code desarrollada por **Yvagacore** para actuar como enlace neural entre el marketplace de **Tepyx** y tu entorno local.

## Vision General
Tepyx es un ecosistema de orquestación neural descentralizada. Esta extensión permite que los diseñadores de arquitecturas IA en Tepyx sincronicen topologías complejas directamente con un motor de inferencia local alimentado por modelos de última generación.

## Funcionalidades Key
- **Neural Sync Protocol**: Sincronización en tiempo real de nodos (Shards) y conexiones mediante WebSockets (Puerto 3010).
- **Proactive Architect**: Integración nativa con Gemini 2.5 Flash para generar lógica industrial (Context -> Agent -> Actuator) a partir de notas estratégicas.
- **Secure Link**: Blindaje de orígenes para dominios autorizados de Yvagacore.

## Requisitos
- Contar con una cuenta activa en [Tepyx](https://tepyx.yvagacore.com).
- Una **Google Gemini API Key** (se te solicitará al iniciar la extensión por primera vez).
